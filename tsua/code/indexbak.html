Installing zshipu onto an HTML Website

  • Just Copy ...

    Here's your unique Code snippet. Click this code to copy it right to your clipboard.

    <!-- Go to www.zshipu.com/dashboard to customize your tools --> <script type="text/javascript" src="//s7.zshipu.com/js/300/zshipu_widget.js#pubid=ra-5db42fab89870e49"></script>

  • ... & paste.

    Paste the above code right above the closing</body> of the HTML in your website on every page you want zshipu to work.

    <html><head><title></title></head><body><h1>Where to install code</h1><p>It should be right before the closing body tag.</p>Paste Here</body></html>

    Setup Inline Tools

    If you'd like to activate an inline tool or Tip Jar, add this additional code within the body of your page. Copy and paste the code below to the places on your page where you want the tools to appear.

    <!-- Go to www.zshipu.com/dashboard to customize your tools --> <div class="zshipu_relatedposts_inline"></div>
