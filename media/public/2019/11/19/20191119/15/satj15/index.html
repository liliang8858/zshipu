    C++ 项目

    1、Snake：贪吃蛇游戏 AI 版，通过算法实现让小蛇通过吃豆，最后蛇的身体填满整个地图算结束。该项目详细描述实现思想以及相关算法的讨论

    🔙 返回目录 🔙


    CSS 项目

    2、MDUI：MDUI 是一套用于开发 Material Design 网页的响应式前端框架。没有任何依赖，支持主题切换，轻量级，低学习成本，文档

    🔙 返回目录 🔙


    Go 项目

    3、aliyungo：非官方的 Aliyun Go语言 SDK 支持API：ECS, OSS, DNS, SLB, RDS, RAM, MNS, STS, SLS, MQ, Push, OpenSearch, DM, Container Service

    4、conference：Go 语言实际项目应用的技术分享

    🔙 返回目录 🔙


    Java 项目

    5、FunGameRefresh：好玩的下拉刷新控件

    6、ProgressManager：一行代码即可监听 App 中所有网络链接的上传以及下载进度，包括 Glide 的图片加载进度。实现原理类似 EventBus 你可在 App 中的任何地方，将多个监听器以 URL 地址作为标识符，注册到本框架。当此 URL 地址存在下载或者上传的动作时，框架会主动调用所有使用此 URL 地址注册过的监听器，达到多个模块的同步更新

    🔙 返回目录 🔙


    JavaScript 项目

    7、veneno：一个基于 Node.js 编写的 web 安全漏洞自动化扫描框架

    8、xdomain：纯 JavaScript 实现 CROS 的库，在线示例

    9、font-spider：字蛛是一个智能 WebFont 压缩工具，它能自动分析出页面使用的 WebFont 并进行按需压缩

    10、slick：实现了几乎所有效果的轮播图插件，在线演示

    🔙 返回目录 🔙


    Objective-C 项目

    11、spectacle：OS X 系统下的窗口管理工具，通过快捷键方便、快捷的调整窗口大小和位置

    12、FLEX：用于 iOS 开发的一组应用内调试工具，功能强大且多，多到不一一列举了

    🔙 返回目录 🔙


    Python 项目

    13、musicbox：基于 Python 编写的网易云音乐命令行版本，使用起来简单优雅，能够快速安装及使用

    14、django-blog-tutorial：基于最新版 Django 1.10 和 Python 3.5，通过 26 篇教程一步步带你使用 Django 从零开发一个个人博客系统，在实践的同时掌握 Django 的开发技巧，完成效果展示

    15、aredis：一款基于 Python3 asyncio 的异步 redis 客户端，支持对于单实例，连接池， 哨兵以及集群。作者希望可以找到志同道合的小伙伴集思广益，一起维护、优化。示例代码如下：

    16、freezegun：时间漫步模块，模拟到某一个时间，使用简单方式多样，实现了装饰器、上下文等调用方式。示例代码如下：

    🔙 返回目录 🔙


    Ruby 项目

    17、mastodon：基于 Ruby 语言的社交网站服务器端所有的源代码，通过这个项目，你可以自己部署一个属于自己的社交网站

    🔙 返回目录 🔙


    其它

    18、vim-galore-zh_cn：Vim 从入门到精通

    19、Spacemacs-rocks：用 21 天学习 Emacs 以及 Spacemacs（Emacs 的配置文件）的使用

    20、SpaceVim：一个社区驱动的模块化 vim/neovim 配置集合，其中包含了多种功能模块，并且针对 neovim 做了功能优化。spacevim 有多种功能模块可供选择，支持多种语言。用户只需要选择需要的模块，就可以配置出一个适合自己的开发环境

    21、english-level-up-tips-for-Chinese：如何提高英语技能

    22、ch：类似 virtualenv，可以在 Linux 下建立虛拟的 home 目录並切换，以管理不同工作或项目的文件
